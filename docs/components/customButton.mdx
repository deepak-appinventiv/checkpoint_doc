# Custom Button

The `Button` component is a dynamic and adaptable button designed for application. It includes features such as loading indicators, customizable styling, and comprehensive accessibility support, ensuring an enhanced user experience.

## Usage

### Basic Example

```javascript
import React from 'react';
import CustomButton from './CustomButton';

const MyComponent = () => {
  const handlePress = () => {
    console.log('Button Pressed');
  };

  return (
    <CustomButton title="Click Me" onPress={handlePress} isLoading={false} />
  );
};

export default MyComponent;
```

## Props

| Prop Name                | Type        | Default         | Description                                             |
| ------------------------ | ----------- | --------------- | ------------------------------------------------------- |
| `title`                  | `string`    | `''`            | Text displayed on the button.                           |
| `disabled`               | `boolean`   | `false`         | Disables the button when `true`.                        |
| `children`               | `ReactNode` | `undefined`     | Child elements to render inside the button.             |
| `testID`                 | `string`    | `''`            | Test ID for the button (useful for testing purposes).   |
| `onPress`                | `function`  | `() => {}`      | Callback function triggered when the button is pressed. |
| `activeOpacity`          | `number`    | `0.7`           | Opacity level for button press feedback.                |
| `isLoading`              | `boolean`   | `false`         | Shows a loading spinner when `true`.                    |
| `spinnerColor`           | `string`    | `colors.white`  | Color of the loading spinner.                           |
| `customStyle`            | `ViewStyle` | `undefined`     | Custom style for the button container.                  |
| `height`                 | `ViewStyle` | `normalize(56)` | Custom height for the button.                           |
| `customButtonTitleStyle` | `TextStyle` | `undefined`     | Custom style for the button title.                      |
| `isShadowRequired`       | `boolean`   | `true`          | Adds shadow styling when `true`.                        |

## Features

- **Loading State:** Displays a spinner when the `isLoading` prop is `true`.
- **Accessibility:** Includes `accessibilityRole` and `accessibilityLabel` for better screen reader support.
- **Customizable Styles:** Allows overriding of default styles with `customStyle` and `customButtonTitleStyle` props.
- **Shadow Support:** Adds shadow styling if `isShadowRequired` is `true`.

## Default Props

The following default props are provided:

```javascript
CustomButton.defaultProps = {
  isLoading: false,
  title: '',
  testID: '',
  disabled: false,
  onPress: () => {},
  activeOpacity: 0.7,
  textTestId: 'button',
  height: normalize(56),
  textColor: colors.white,
  fontSize: normalize(15),
  medium: 'medium',
};
```

## Styling

The component leverages predefined styles in the `styles` file and `commonStyles.shadow`. You can customize these using the `customStyle` and `customButtonTitleStyle` props.

### Example: Custom Styling

```javascript
<CustomButton
  title="Styled Button"
  customStyle={{ backgroundColor: 'blue', borderRadius: 10 }}
  customButtonTitleStyle={{ fontSize: 18, color: 'yellow' }}
  onPress={() => console.log('Custom Styled Button Pressed')}
/>
```
