# HomeHeader

The `HomeHeader` component is a customizable header designed for the home screen of our app. It features a background image, user information, and an interactive right icon button, making it versatile and visually appealing.

## Props

| Prop                  | Type          | Default                | Description                                                 |
| --------------------- | ------------- | ---------------------- | ----------------------------------------------------------- |
| `onRightIconPress`    | `Function`    | `null`                 | Callback function triggered when the right icon is pressed. |
| `rightIconStyle`      | `Object`      | `{}`                   | Custom styles for the right icon.                           |
| `rightContainerStyle` | `Object`      | `{}`                   | Custom styles for the right icon container.                 |
| `bgImage`             | `ImageSource` | `images.headerBgImage` | Background image for the header.                            |
| `rightIcon`           | `ImageSource` | `icons.cloudIcon`      | Icon displayed on the right side of the header.             |
| `userName`            | `string`      | `''`                   | User's name displayed below the date and update section.    |
| `headerColor`         | `string`      | `'#000'`               | Background color for the "Last Updated" badge.              |
| `formatDateTime`      | `string`      | `''`                   | Formatted date and time string displayed at the top left.   |

## Usage

### Basic Usage

```jsx
import React from 'react';
import HomeHeader from './HomeHeader';
import { Alert } from 'react-native';

const App = () => {
  return (
    <HomeHeader
      userName="John Doe"
      formatDateTime="December 23, 2024, 10:00 AM"
      headerColor="blue"
      onRightIconPress={() => Alert.alert('Right icon pressed!')}
    />
  );
};

export default App;
```

### Customizing Appearance

```jsx
<HomeHeader
  userName="Jane Smith"
  formatDateTime="December 23, 2024, 2:30 PM"
  headerColor="green"
  bgImage={require('./assets/customHeaderImage.png')}
  rightIcon={require('./assets/customRightIcon.png')}
  rightContainerStyle={{ padding: 10 }}
  rightIconStyle={{ width: 24, height: 24 }}
  onRightIconPress={() => console.log('Custom right icon pressed')}
/>
```

## Features

- **Background Image**: Customize the header with a background image.
- **User Info Display**: Displays the user's name prominently.
- **Last Updated Section**: Highlights the date and time of the last update with a badge.
- **Right Icon**: An interactive icon on the right-hand side for additional functionality.
- **Safe Area Insets**: Automatically adjusts padding to accommodate device notches.

## Notes

- The component is optimized for different screen sizes using responsive dimensions.
- The `Bounceable` wrapper provides a smooth animation for the right icon interaction.
